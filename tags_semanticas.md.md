<header>
O header é utilizado para representar o cabeçalho de um documento ou seção declarado no HTML.
 Nele podemos inserir elementos de <h1> a <h6>, até elementos para representar imagens, parágrafos ou mesmo listas de navegação.

<section>
O elemento section representa uma seção dentro de um documento.

<article>

Article é utilizado para declarar um documento que não precisa  de outro para fazer sentido.

<nav>
Nav> é utilizado quando precisamos representar um agrupamento de links de navegação.

<aside>
Aside é utilizado quando precisamos criar um conteúdo de apoio/adicional ao conteúdo principal.

<main>
Main especifica o conteúdo principal.

<figure>
O elemento <figure> é uma marcação para a inserção de uma figura.

<footer>
O elemento <footer> representa um rodapé de um documento.

<a>
A principal função do elemento <a> é descrever um link.
<em>
O elemento <em> é utilizado quando desejamos enfatizar um trecho ou palavra no texto

<strong>
O elemento <strong> também é utilizado para destacar uma parte do texto.

<cite>
O elemento <cite> é utilizado para declarar que naquele trecho há uma citação.


<time>
O elemento <time> é utilizado para representar datas.
